# Анализ поиска в Яндекс Картинках

## Описание
Анализ запросов к Яндекс Картинкам за сентябрь 2021 года. Сравнение поведения пользователей на desktop и touch.

## Содержимое
- `analysis.ipynb` — Jupyter Notebook с полным анализом
- `data.tsv` — исходные данные (семпл запросов)
- `datalens_dashboard.png` — скриншот дашборда в DataLens

## Выводы
- На десктопах преобладают рабочие и образовательные запросы
- На мобильных устройствах — поздравления, развлечения, мода
- Пик активности на desktop — 13:00, на touch — 17:00 и утром

## Стек
- Python (pandas, matplotlib, seaborn)
- SQL (pandasql)
- Yandex DataLens
